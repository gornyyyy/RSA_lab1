### Шифрование
- текст переводится в байты
- затем байты разбиваются на блоки
- с помощью C ≡ M^S (mod N), т.е. Big.Integer.ModPow(m, s, N) блоки шифруются

Размер блока зависит от размера N  
### Расшифровка
- зашифрованные блоки (через пробел) переводим в расшифрованные блоки с помощью M ≡ C^E (mod N), т.е. Big.Integer.ModPow(с, e, N)
- переводим числа в байты, а затем в текст по кодировке UTF-8
